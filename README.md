# vityarthi-project
Gift Management Inventory is a Python-based command-line system for tracking gift shop stock. It allows adding, removing, updating, and searching items by ID, displaying prices and availability. Uses dictionaries and loops for efficient inventory management.
Gift Management Inventory System

Project Title: Gift Management Inventory

Overview:
A Python-based command-line inventory management system designed for gift shops to efficiently track stock levels, manage product information, and retrieve item details quickly using unique ID codes.

Features:

Add new gift items with ID, name, price, and stock

Check item availability by ID

Retrieve item prices instantly

Update stock levels (increase/decrease)

Remove discontinued items

Display complete inventory list

Clear entire inventory

Technologies/Tools Used:

Python 3.x

Built-in data structures (dictionaries, lists)

Standard input/output handling

Control structures (loops, conditionals)

Steps to Install & Run:

Install Python 3.x from python.org

Save the code as gift_inventory.py

Open terminal/command prompt

Navigate to file location

Run: python gift_inventory.py

Instructions for Testing:

Run the program

Use menu option 6 to view initial inventory

Test adding new items (option 1)

Check availability (option 2) with ID 101

Retrieve price (option 3) with ID 102

Update stock (option 4) - try both increase and decrease

Remove item (option 5)

Clear inventory (option 7)

Exit using option 0
